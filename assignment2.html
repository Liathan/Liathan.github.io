Assignment 2: Distributions and correlations

Histogram of tree size

Using the histogram, the user can visualize the distribution of a certain measurement of the trees in the whole dataset. The user can also choose the range of compression of the bins to consider for the visualization of the histogram. The general behaviour of the distributions seems to follow an asymmetrical and right skewed shape.
It is possible to visualize the range and the amount of trees in each bin overing it with the mouse.

Boxplot of top 5 tree species size

Considering the same measurements saw before, now we visualize the boxplots of the top 5 tree species. Thanks to boxplots, we can display some descriptive statistical informations like the median, the maximimum, the minimum, and the quartiles of each distribution. Potentials outliers are plotted as individual points. For example, looking at the height, we can instantly see that Platanus x hispanica species is usually higher with respect to the other species, and more spread, lookind at the IQR.
It is possible to visualize the Q1, Q3, mean, and median values of each box overing it with the mouse.

Scatterplot of a specific measure and the absorbed CO₂

For the top 6 most populated tree species, we display a scatterplot having on the x axis the size of trees, like the height, diameter, canopy size, and leaf area, and and on the y axis the kilograms of CO₂ absorbed in a year. We want to discover possible correlations between these variables.
Moving the mouse over the single data-point, we can visualize the exact value of the selected measure and the amount of CO₂ absorbed by the tree, highlighting all the trees of this species.

Small multiple scatterplot of different measures and the absorbed CO₂

The visualization via multiple small scatterplot, as it separates the tree species, help us to see the different distrubutions of the points based on the species. For example we can now notice how the Tilia x europaea and the Celtis australis beahaviors on absorbing CO₂ are different, changing the tree height.
Moving the mouse over the single data-point, we can visualize the exact value of the selected measure and the amount of CO₂ absorbed by the tree, highlighting all the trees of this species.

Bubblechart of a specific measure and the absorbed CO₂, exploring the canopy size

With the bubblechart, we can analyze the same things of the third plot, but we can also add another information, visualized through the size of the bubbles which, in this case, represents the canopy size of the tree.
It is possible to see the specific canopy size of a tree and other informations, overing the corresponding bubble with the mouse.
